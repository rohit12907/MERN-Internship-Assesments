# React + Vite Project

This project is built using **React** and **Vite** to create a fast and responsive web application. Vite provides a lightweight development environment with Hot Module Replacement (HMR), making development faster and smoother.

## 🚀 Tech Stack

- React.js
- Vite
- JavaScript (ES6+)
- HTML5
- CSS3

## 📂 Project Setup

### Clone the repository

```bash
git clone <your-repository-url>
```

### Move into the project folder

```bash
cd <project-folder-name>
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```

The application will be available at:

```
http://localhost:5173
```

## 📦 Build for Production

```bash
npm run build
```

## 👀 Preview Production Build

```bash
npm run preview
```

## ✨ Features
